# Draft IPDB Theme Hierarchy

One-level parent/child hierarchy for pinball machine themes.
Each machine gets tagged with the most specific (child) term; the parent is implied.
Themes marked ~~strikethrough~~ are dropped — they're metadata, not themes.

Machine counts from IPDB in parentheses.
